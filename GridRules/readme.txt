The purpose of GridRules:
    -contains scripts which have rules for the purpose of manipulating the simulation in different ways and increasing flexibility


Default:
    -contains the rules for the default game of life

Simple:
    -contains simple rules that can be utilised in more complex rules

Tools:
    -contains all the functionality to be utilised in building versatile rulesets and simple rulesets
    -this is utilised by every other script within GridRules

Chess:
    -contains rulesets that are inspired by Chess such as the knight
